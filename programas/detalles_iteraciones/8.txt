bloque de cada experimento

    nombre del experimento

    si hubo `class_weight` o no
    distribucion original del train

epocas de entrenamiento
    en cada `Epoch` se muestran:
    `accuracy`
        porcentaje total de aciertos
    `loss`
        funcion de perdida durante entrenamiento
    `val_accuracy`
        accuracy sobre validacion
    `val_loss`
        loss sobre validacion

evaluacion por threshold
    para cada threshold se imprime:
    `precision`
        de todos los casos que el modelo marco como fraude, cuantos realmente eran fraude
    `recall`
        de todos los fraudes reales, cuantos encontro el modelo
    `f1`
        equilibrio entre precision y recall

matriz de confusion
    se imprime como:
    `TN`: verdaderos negativos
        no fraude predicho como no fraude
    `FP`: falsos positivos
        no fraude predicho como fraude
    `FN`: falsos negativos
        fraude predicho como no fraude
    `TP`: verdaderos positivos
        fraude predicho como fraude

ejemplos de `FormattedText`

ejemplo fuerte de matriz de confusion:

resultado:
    accuracy alta, pero totalmente engañosa

    este experimento puede verse como una linea base
